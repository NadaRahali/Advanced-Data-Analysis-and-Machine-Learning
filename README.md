# Deep Learning – Period 2 Exercises

This repository contains my individual exercise solutions for the Deep Learning (Period 2) course.
The exercises correspond to the weekly topics outlined in the course plan provided by the professor.

## Repository Structure

```
├── week1/          # Week 1 exercises
├── week2/          # Week 2 exercises
├── week3/          # Week 3 exercises
├── week4/          # Week 4 exercises
├── week5/          # Week 5 exercises
├── week6/          # Week 6 exercises
├── requirements.txt # Python dependencies
└── README.md       # This file
```

## Setup

To set up the environment for running these exercises:

```bash
pip install -r requirements.txt
```

## Weekly Topics

- **Week 1**: Introduction to Deep Learning
- **Week 2**: Neural Network Fundamentals
- **Week 3**: Convolutional Neural Networks (CNNs)
- **Week 4**: Recurrent Neural Networks (RNNs)
- **Week 5**: Advanced Architectures
- **Week 6**: Optimization and Regularization

Each week's directory contains:
- Exercise notebooks (`.ipynb` files)
- Python scripts (`.py` files)
- Data files (if applicable)
- README with specific exercise descriptions
